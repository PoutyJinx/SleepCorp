# SleepCorp

SIN Corporation themed **sleep tracking cog** for **Red-DiscordBot**.

This cog allows users to log when they go to sleep and when they wake up, while generating humorous **SIN Corporation HR evaluations** based on their sleep duration.

Instead of boring sleep logs, SleepCorp turns your community's sleep habits into a **corporate incident report from the depths of SIN Corp**.

---

## Features

• `/sleep` — Log when you went to bed  
• `/awake` — Log when you woke up and receive your sleep evaluation  
• `/bedtime` — Check the last recorded bedtime for a user  

The bot calculates:

- total sleep duration
- a **0–5 star sleep rating**
- randomized **SIN Corporation HR commentary**

All responses are written in **SIN Corporation style**, including HR commentary, corporate evaluation lines, and sarcastic sleep reports.

## Installation

```
[p]repo add sleepcorp https://github.com/PoutyJinx/SleepCorp
[p]repo update sleepcorp
[p]cog install sleepcorp sleepcorp
[p]load sleepcorp
[p]slash sync
```

If slash commands are disabled:

```
[p]slash enable sleep
[p]slash enable awake
[p]slash enable bedtime
[p]slash sync
```

---

## Requirements

- Red Discord Bot V3
- Python 3.11+
- Slash commands enabled
